Loading Grandivory's Mods Optimizer for Star Wars: Galaxy of Heroes™

Oh no! If this text is up long enough for you to read it, it means that something went wrong. Most likely, your browser is trying to access an old version of the javascript that runs this site, but it doesn't exist anymore. If that's the case, then you need to clear your browser's cache and reload, and everything should work again. Exactly how you do that, though, depends on your browser. Try the instructions below, but if they don't work, feel free to ask for help on the Discord server.

Please note: Grandivory's Mods Optimizer is tested to work in Firefox, Chrome, and Safari on desktop only! Other browsers may work, but they are not officially supported. If you're having trouble, try using one of the supported browsers before asking for help.

Google Chrome

  1. Open up dev tools: hit 
     f12
    on Windows or 
     Cmd + Option + I
    on a Mac
  2. Go to the 
     Console
    tab to check for any error messages.
  3. Go to the 
     Application
    tab.
  4. Select 
     Clear storage
    from the sidebar.
  5. Check 
     Cache storage
    and 
     Application cache
    and hit 
     Clear site data
    .
  6. Reload the page.

Firefox

In firefox, you can either completely forget about a particular site (removing all saved data about that site), or you can clear the cache for all sites, but you can't just clear the cache for one site.

Forget the Mods Optimizer

  1. From the menu, select 
     Library > History
    then click 
     Show All History
    at the bottom.
  2. Find the Mods Optimizer in the list of sites visited.
  3. Right click on the Mods Optimizer and select 
     Forget about this site
    .
  4. Reload the page.

Clear Firefox's Application Cache

  1. From the menu, select 
     Preferences
    .
  2. Select 
     Privacy & Security
    from the sidebar.
  3. Click on 
     Clear data
    under the 
     Cookies and Site Data
    heading.
  4. Select 
     Cached Web Content
    .
  5. Hit 
     Clear
    .
  6. Close preferences and reload the page.
